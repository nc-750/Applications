# Changelog

## 0.1.0 (unreleased)

Initial version.

- **Interview:** streaming chat interview with free (2–3 questions) and pro (5–8 questions) tiers.
- **Synthesis:** structured output with fallback to plain-text parsing. Validated against Zod schema.
- **Insight:** private self-knowledge document — strengths with evidence, growth areas, hidden assets, personality dimensions, values, goals, ready-to-use text.
- **Profile:** public shareable page — strengths, skills, timeline, beyond work, "How I Work Best" section.
- **Providers:** OpenAI, Anthropic, Mistral, and any OpenAI-compatible endpoint (Groq, Together, OpenRouter, Ollama, LM Studio).
- **Platforms:** Windows, macOS, Linux (Tauri desktop) + installable PWA.
- **Security:** OS keyring for API keys on desktop, strict CSP, self-contained HTML exports.
- **Licensing:** Lemon Squeezy integration for Pro tier.
- **Data management:** import/export mirror.json, three-tier wipe controls, factory reset.
- **Design:** Warm Midnight / Aged Vellum dual theme with terracotta accent. Responsive layout with sidebar (desktop) and bottom nav (mobile).
