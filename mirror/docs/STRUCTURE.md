# Mirror Structure

mirror/                     # Root folder for the Mirror project
├── app                     # Root folder for the Mirror application
├── deployed-app            # Built version of the Mirror app, ready to be deployed.
├── docs                    # Documentation for the Mirror project
│   └── to_be_reworked      # Outdated documents that still have useful data, but needs a rework to be used for production. Kept as reference
├── resources               # Other files and projects that do not have a clear category and that can be shared by the other projects inside of the Mirror folder
└── website                 # Presentation website for the Mirror project
